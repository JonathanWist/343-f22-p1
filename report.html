Technical Components

  1. There are 5 pages on the site (home, movement, centralrock, brooklyn, and info) with home being on index.html. report.html is present in the folder, but is not on the site.
  2. Each page uses the same favicon named in the files favioc.ico (including this page).
  3. The site has a header on every page.
  4. The site has a navbar on every page.
  5. There is support for small and large screens (see comments in the css/style.css file to see where it happens).
  6. There are many images and all of them have an alt property defined with a descriptive text associated with each.
  7. Each page validates and passes the accessibility tests.
  8. My site uses 2 different font-faces (header uses "Segoe UI" which is in the Sans-Sarif family, and p elements use "Georgia" in the Sarif family).
  9. My site contains 1 form on the movement page.
  10. All 3 pages contain an external link and the info page contains several links to external sites.
